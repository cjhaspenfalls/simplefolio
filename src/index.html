This website uses the Clarifai Face Recognition API to detect faces in JPEG images. This is a Full Stack project consisting of a Front-End using React.js, a Back-End server using Node.js, and a SQL server database. Sign up and upload your JPEG URLs to get started!

Gradient Background Generator

On this website, the user selects two colors of their choice and a beautiful color gradient is generated. This was developed strictly using HTML, CSS, and a bit of JavaScript DOM Manipulation.

The site will even display the CSS code of the background so the user can copy and paste it into their own projects! Having trouble deciding what colors to pick? Click the "Randomize" button and let the website decide for you!

RoboFriends

Built using React.js. This website fetches information from JSON Placeholder to simulate a user database. It uses the Robohash API to generate unique Robot profile images. The user can search the list of RoboFriends. Not case sensitive. The app is responsive so that when viewed on a mobile device or small window, the list is scrollable while still having access to the search box.
