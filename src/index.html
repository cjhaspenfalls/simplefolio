Face Recognizing Brain

This website uses the Clarifai Face Recognition API to detect faces in JPEG images. Simply sign up, upload your images, and watch the brain work its magic! The user can sign back in anytime using their credentials. The site even keeps track of how many images the user has uploaded so far!

This is a Full Stack project consisting of a Front-End built with React.js, a Back-End server using Node.js, and a SQL server database. For added security, an API is used that converts the user's password to a hash, so even the owner of the site cannot steal anyone's password.

Sign up and upload your JPEG URLs to get started!

Gradient Background Generator

RoboFriends

This website allows the user to filter a list of robots using the search box. The names and email addresses of the robots are fetched from JSON Placeholder to simulate a real-world user database. The site uses the Robohash API to generate the unique robot profile images.

The search box is not case sensitive, allowing for a more enjoyable user experience. The site is even responsive so that when viewed on a small browser window or mobile device, the list becomes scrollable while still giving the user access to the search box.

Give it a go! It's time to meet your new friends!
